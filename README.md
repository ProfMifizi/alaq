ALAQ - Déploiement Vercel (phase test HTML statique)

Contenu :
- index.html : application prototype ALAQ en HTML/CSS/JS autonome

Méthode la plus simple sans GitHub :
1. Aller sur https://vercel.com/drop
2. Glisser-déposer ce dossier ou ce ZIP
3. Choisir un nom de projet, par exemple alaq-app
4. Cliquer sur Deploy
5. Tester l'URL fournie par Vercel

Méthode recommandée ensuite :
- Créer un dépôt GitHub
- Mettre index.html à la racine du dépôt
- Importer le dépôt dans Vercel
- Chaque modification poussée sur GitHub redéploiera automatiquement le site

Domaine :
- Garder le domaine chez Squarespace
- Relier idéalement app.votredomaine.com à Vercel dans un premier temps
